WHICH TYPE OF WEBSITE I NEED?

We're here to talk you through that process. We'll gather some information about the purpose of your website and offer the ideal approach. Below is a brief outline of the common types of websites we'll be developing.

STATIC WEBSITES

Pros:
  • Low development costs
  • Simpler hosting needs
  • Faster, lighter and easier to maintain
Cons:
  • Developer needed for content updates
  • Search Engine Optimization (SEO) favours fresh content
  • Limited features

Static Websites are fixed content-type websites where the content does not need to be frequently changed. These are a good, light solution to show information and give you a web presence. Usually built with HTML, CSS and Javascript.

CMS / ECOMMERCE WEBSITES

Pros:
  • Easily modify your content
  • More powerful features
  • Search Engine Optimization (SEO) benefits
Cons:
  • More complex to develop and host
  • Slower load times due to running a database
  • Requires higher maintenance and security

Content-Management System allows you to log into an admin panel and update particular parts of the content without programming knowledge. These dynamic websites are popular among bloggers and users who need to update the content on a regular basis.
For those looking to sell products online, these websites are built off CMS websites but with the additional functionality to manage products and receive payments online. Usually built with Wordpress or Shopify.
